# JS Event Processor

Mini-projeto em JavaScript focado em fundamentos da linguagem:

## Tópicos abordados
- var, let e const
- Tipos primitivos
  - string
  - number
  - boolean
  - null
  - undefined
  - symbol
  - bigint
- Conversão e coerção de tipos

## Objetivo
Processar uma lista de eventos com dados inconsistentes, normalizando tipos
e gerando um relatório final.

## Execução
```bash
node src/index.js
```
